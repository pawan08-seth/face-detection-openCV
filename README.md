# Real-Time Face Detection using OpenCV

This project performs real-time face detection using OpenCV and Haar Cascade classifier.

## 📌 Features
- Detects faces using webcam
- Draws bounding box around detected faces
- Real-time processing
- Uses pre-trained Haar Cascade model

## 🛠 Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## ▶ How to Run

1. Install OpenCV:
   pip install opencv-python

2. Run the script:
   python face_detection.py

Press 'q' to exit the webcam.
